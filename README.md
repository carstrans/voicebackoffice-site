# Voice Back Office landing page

Статический лендинг для заявки на презентацию голосового ИИ-продавца.

Откройте `index.html` или запустите локальный сервер:

```bash
python3 -m http.server 8000
```

Форма сейчас демонстрационная: заявки сохраняются в `localStorage`. Перед публикацией подключите рабочий endpoint формы или CRM/webhook.
